\item d) 被5或7整除？

\end{description}

\section*{Problem 3}
在一个婚礼上摄影师从10个人中安排6个人在一排拍照，其中新娘和新郎在内，如果满足下述条件，有多少种安排方式？
\begin{enumerate}[a)]
\item 新娘必须在照片中
\item 新娘不在新郎旁边
\item 新娘在新郎左边的某个位置
\end{enumerate}

\item 她必须选多少个球才能保证至少有3个球是蓝色的？
\end{enumerate}

\section*{Problem 5}
证明：如果$f$是从$S$到$T$的函数,其中$S$和$T$是有穷集，满足$|S|>|T|$，那么在$S$中存在元素$s_1$和$s_2$使得
$f(s_1)=f(s_2)$，或者换句话来说，$f$不是一对一的。

\section*{Problem 12}

\section*{Problem 13}
一所学校的数学系有7名女教师和9名男教师。
\begin{enumerate}[a)]
\item 有多少种方式从中选出5人的委员会并使其中包含至少1名女教师？
\item 有多少种方式从中选出5人的委员会并使其中包含至少1名女教师和至少1名男教师。
\end{enumerate}

\section*{Problem 14}
一个新月形面包店有普通新月形面包、樱桃新月形面包、巧克力新月形面包、杏仁新月形面包、苹果新月形面包和椰菜新月形面包。有多少种选择方式
\begin{enumerate}[a)]
\item 12个新月形面包？
\item 36个新月形面包？
\item 24个新月形面包，并且至少每类有2个？
\item 24个新月形面包，并且不超过2个椰菜的？
\item 24个新月形面包，并且至少5个巧克力的且至少3个杏仁的？
\item 24个新月形面包，并且至少1个普通的，至少2个樱桃的，至少3个巧克力的，至少1个杏仁的，至少2个苹果的和不超过3个椰菜的？
\end{enumerate}

\section*{Problem 20}

\end{enumerate}
